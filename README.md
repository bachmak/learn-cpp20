This is my learning journey with the [Programming with C++20](https://www.amazon.de/Programming-20-Concepts-Coroutines-Ranges/dp/3949323015) book by Andreas Fertig.
Here, I follow along the chapters and experiment with new features in a meaningful (but sometimes also in an absurd) way. I reflect my notes/observations in the comments.

Where possible, I'm striving to make the whole thing work at compile time by implementing compile-time unit tests. The empty main functions are there to simply make the linker happy.
